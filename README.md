# Passenger Information Display System (PIDS)

Ein leichtgewichtiges REST-API-Backend zur Verwaltung und Anzeige von Zugfahrplänen und Abfahrtszeiten. Entwickelt als Demonstration moderner Backend-Architektur mit Spring Boot.
Ein sehr primitive Version von PIDS, die man an Bahnhofen oder Haltestellen sowie Stationen täglich sehen kann.

---

## Tech Stack

* **Spring Boot**
* **Spring Data JPA** & **PostgreSQL**
* **Jakarta Validation** (Eingabevalidierung)
